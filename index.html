RLDP is a simple, scalable, and robust representation method that uses a latent prediction objective with diversity regularization for strong zero-shot RL.

Talk and Overview

Does latent next-state prediction produce state features that enable performant zero-shot RL?

Using naive latent next-state prediction can cause a mild form of collapse, where representations of different states become increasingly similar over training.

How it works?

RLDP combines latent prediction objective with orthogonality regularization to encourage feature diversity and learn more diverse representations.

The goal of RLDP is to learn features both predictive of future state evolution and diverse enough to support a broad family of rewards.

Results

Offline Zero-Shot RL
We consider continuous control tasks from DeepMind control suite -- Pointmass, Cheetah, Walker, Quadruped under a similar setup considered by prior works in zero-shot RL to benchmark RLDP with respect to state-of-the-art methods. We use datasets from the ExoRL suite that are obtained by an exploratory algorithm RND . Random features use representations from a randomly initialized NN encoder.
Online Zero-Shot RL
We also explore if the learned representation enable competitive multi-task learning when agent is allowed interaction with the environment. For this experiment, we experiment in the high-dimensional SMPL humanoid environment and perform evaluation on 45 different tasks. The representations are trained offline using a fixed dataset. Then, we freeze the representations and do the policy learning online with an oracle policy. The violin plot having the mean above 0 indicates that RLDP is mostly competitive and sometimes better than baseline representation learning methods.
Learning Representations with Low Coverage Datasets
RLDP learns a policy-independent representation through latent dynamics prediction. To understand how these representations are impacted by offline data coverage, we evaluate RLDP and baseline methods in the D4RL benchmark of OpenAI Gym MuJoCo tasks. The violin plot shows one-to-one comparison of normalized returns of RLDP against baseline methods. The overall mass of the violin lies above zero, indicating that RLDP outperforms baseline methods in many of these tasks.
